## 核心发现速览

### Spielberg 的创意工作流
```
直觉感知 → 简化框架 → 现场协作 → 情感真理表达
```

### 视觉美学三支柱
1. **L系统** - 垂直+线性运动的交替
2. **Spielberg Face** - 凝视心理学
3. **阻挡设计** - 身体位置的道德表达

### 思想演进三阶段
```
1975-1985: 冒险与逃脱期（娱乐优先）
    ↓
1993-2005: 道德觉醒期（Schindler's List转折）
    ↓
2012-2026: 自传与反思期（脆弱性、根源性）
```

### 一个核心真理
> 最好的电影，既是个人的，也是普遍的；  
> 既是商业的，也是艺术的；  
> 既是感性的，也是伦理的。
