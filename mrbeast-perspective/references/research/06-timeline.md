# MrBeast 完整时间线（1998-2026）

> James Stephen "Jimmy" Donaldson — 从堪萨斯州威奇托的男孩到地球上最大的 YouTuber 与娱乐工业家。
> 本时间线聚焦"思想转折点"和最近12个月的动态。

---

## 主时间线表

| 年份 | 事件 | 思想转折 / 重要性 |
|------|------|-------------------|
| **1998.5.7** | 生于堪萨斯州 Wichita，本名 James Stephen Donaldson | 起点。母亲 Sue Parisher，父亲 Charles Donaldson。家中第二子，哥哥 CJ（后成为 YouTuber "MrBro"），还有一个弟弟 |
| **1998-2005** | 7 岁前家庭迁居 3 个城市 | 频繁搬家造就内向性格，母亲 Sue 后来认为这是他童年最重要的塑造因素之一 |
| **2007** | 父母离婚；与母亲及哥哥定居北卡州 Greenville | 与父亲断绝联系（至今）。母亲 Sue 后来披露在婚姻中遭受身体、性、情感虐待。母亲成为他唯一的"原家庭"，也是他后来商业帝国的合规官 |
| **2012.2** | 13 岁，上传第一条 YouTube 视频；频道名 "MrBeast6000"（Xbox 随机生成的 gamertag） | **思想起点**：YouTube 不是兴趣，是"职业"。早期内容是 Minecraft、Black Ops 2 的 Let's Play |
| **2012-2014** | 持续低播放量；研究 YouTube 算法 | 与一群朋友（Chris、Chandler、Garrett 等）形成"分析型创作者圈"。每天讨论 YouTube 算法 |
| **2013-2014** | 15 岁前后确诊 Crohn's 病（克罗恩病） | **身体的限制变成精神的训练**：一夏天瘦 30 磅，每天上 8-10 次厕所。学会"忽视身体不适专注于目标"。终身免疫抑制剂 Remicade 治疗，被他描述为"hard mode 模式生活" |
| **2015-2016** | "Worst Intros on YouTube" 系列让他在小圈子走红 | 第一次体会"洞察力 > 内容生产"。到 2016 年中订阅约 30,000 |
| **2016** | 从 Greenville Christian Academy 高中毕业 | 母亲坚持他读大学 |
| **2016 年底** | 入学 East Carolina University，2 周后辍学 | **第一次重大决策转折**：选择 YouTube 而非大学。母亲愤怒，把他赶出家门。"全押 YouTube"的赌局正式开始 |
| **2017.1.8** | "I Counted to 100,000!"（数到 10 万）发布 | **生涯爆发点**：实际花了 40 小时（边数边看 Game of Thrones 和 Naruto）。视频几天内数千万播放。首个病毒视频，奠定"耐力 + 极端 + 完整呈现"内容公式 |
| **2017.5** | 突破 100 万订阅 | 11 个月从 30k 涨到 1M。算法研究的回报 |
| **2017.6** | 收到首个品牌合作（Quidd），用 $10,000 全部送给一个流浪汉 | **思想转折：从"古怪挑战"到"慈善+娱乐"**。发现"给钱"是最有传播力的内容形式 |
| **2018** | 累计已通过视频送出超 100 万美元 | 被媒体冠以"YouTube 最大慈善家"。"1 Billion Orbeez in Friend's Backyard" 等极端实验视频走红 |
| **2018.11.6** | 突破 1000 万订阅 | 18 个月从 1M 到 10M。指数增长开始 |
| **2019.10.25** | 与 Mark Rober 共同发起 **#TeamTrees**（庆祝 20M 订阅） | **第一次大型协调行动**：55 天筹集 $20M（包括 Elon Musk、Jack Dorsey），全部交给 Arbor Day Foundation 种 2000 万棵树。证明"个人 YouTuber 可以驱动企业级公益" |
| **2020.11** | 启动 **Beast Philanthropy** 频道，配套自建食物银行 | **慈善制度化**：100% 频道收入归慈善。任命 Darren Margolias 为执行董事。从"个人壮举"转向"机构化运营" |
| **2020.12** | **MrBeast Burger** 上线（与 Virtual Dining Concepts 合作的虚拟餐厅） | **首次进入实体消费品**：三个月卖出 100 万个汉堡。但后期质量问题埋下 2023 年诉讼的种子 |
| **2021.10.29** | 与 Mark Rober 发起 **#TeamSeas**（庆祝 50M 订阅） | 65 天筹集 $30M，清理 3400 万磅海洋垃圾。Team Trees 模式的复制验证 |
| **2021.11.24** | "$456,000 Squid Game in Real Life!"（真人版鱿鱼游戏）发布 | **生涯最重要的单条视频**：3 天破亿播放，8 天 1.42 亿。制作成本 $350 万。**思想转折：他证明 YouTuber 可以做电视台级别的制作**。这条视频后来成为 Beast Games 的种子 |
| **2022.1.29** | **Feastables** 巧克力品牌发布 | **从代言转向自有品牌**：花一年时间做产品研发；首周 100 万条售出（约 $1000 万收入）。第一年达到 $1 亿销售额，成为美国增长最快的巧克力品牌 |
| **2022.7.28** | 突破 1 亿订阅 | 全球第二人（仅次于 PewDiePie 之外的频道）。首位个人创作者过亿 |
| **2023.1.28** | "1,000 Blind People See For The First Time"（资助 1000 例白内障手术） | **慈善+争议的拐点**：视频 24 小时破 3000 万播放。但也引发"为什么美国医疗体系需要一个 YouTuber 来做这件事"的尖锐批评。Donaldson 第一次被认真讨论为"系统性社会问题的反映" |
| **2023.4** | 长期搭档 Chris Tyson 公开正在做 HRT（激素替代疗法），后来以 Ava Kris Tyson 身份生活 | Jimmy 公开支持。但部分观众的过激反应预示了 2024 年的危机 |
| **2023.7** | 起诉 Virtual Dining Concepts，要求关闭 MrBeast Burger | "保护品牌 > 短期收入"。VDC 反诉 $100M |
| **2023.8** | 与 Amazon MGM Studios 签下 $100M+ 协议制作 Beast Games | **从 YouTuber 到 Hollywood 制片人**的关键一跳 |
| **2023.10.15** | 突破 2 亿订阅 | 15 个月从 100M 到 200M |
| **2024.6.1** | 在订阅数上正式超越 T-Series，成为全球最大 YouTube 频道 | 推文称粉丝"为 PewDiePie 复仇"（2018-19 PewDiePie vs T-Series 之战的延续） |
| **2024.7.10** | 突破 3 亿订阅 | |
| **2024.7.23** | **Ava Kris Tyson 事件爆发**：被指控向未成年人发送不当信息（grooming）；当日辞职 | **生涯最大公关危机**：Jimmy 公开声明"震惊与反对"，雇佣第三方律所开启 3 个月独立调查 |
| **2024.8-10** | 前 Beast Games 选手集体诉讼；前员工 Dawson French（"DogPack404"）发布系列指控视频；Rolling Stone 等媒体深度报道工作环境问题 | **从"独立创作者"到"被审视的雇主"**：被迫面对作为 CEO 的责任问题 |

## 关键思想转折点（蒸馏版）

1. **2012**：把 YouTube 当成"职业"而非"兴趣"——开始算法分析
2. **2016**：辍学 + 被赶出家门——"全押"心智的代价化承诺
3. **2017.1**：数到 10 万——发现"耐力 + 极端 + 真实"是稀缺资源
4. **2017.6**：第一次把品牌钱全部送给流浪汉——发现"慈善是最强内容引擎"
5. **2019.10**：Team Trees——发现自己可以"调度社会资源"
6. **2021.11**：真人版鱿鱼游戏——证明"YouTuber 可以做电视台规模制作"
7. **2022.1**：Feastables——从"卖广告"转向"卖产品"
8. **2023.1**：1000 例白内障手术——意识到自己已是"系统性议题的镜子"
9. **2024.7**：Tyson 危机——被迫从"创作者"成长为"CEO/雇主"
10. **2024.12**：Beast Games 上线 Prime——正式跨入 Hollywood
11. **2026.2**：收购 Step——从"内容/消费品"扩展到"金融基础设施"

## 当前状态（截至 2026 年 5 月）

- **订阅数**：约 4.77 亿（全球第一，仍以约 13 万/日的速度增长）
- **公司估值**：Beast Industries 约 $5 亿至 $5.2B 区间
- **个人净资产**：约 $26 亿（主要为 Beast Industries 股权账面价值）
- **核心业务结构**：
  - YouTube 主频道（旗舰内容）+ Beast Philanthropy（慈善频道）+ Beast Reacts、Beast Games 频道等
  - Feastables（零食消费品，年销 $1 亿+）
  - Lunchly（与 Prime 合作的儿童午餐盒）
  - Beast Games（Amazon Prime 长寿剧，已续订到 S3-S4）
  - Beast Land（沙特主题公园，限期）
  - Step（fintech，刚收购）
- **个人状态**：与 Thea Booysen 订婚（2024.12.25）；右眼接近失明；Crohn's 长期免疫抑制治疗
- **悬而未决的问题**：
  - Beast Games 选手集体诉讼仍在进行
  - 沙特合作带来的长期声誉成本
  - 公司治理、工作文化是否真正改革
  - 从"创作者"到"娱乐工业 CEO"的身份转换是否完成
